## 🚀 Inicio Rápido

### 🧱 Prerrequisitos

- Node.js >= 16
- npm / pnpm / yarn

### 🛠️ Instalación

```bash
git clone https://github.com/Vann06/miniso-ecommerce.git
cd miniso-ecommerce
```

### ▶️ Ejecutar localmente

```bash
cd server
npm install
npm run dev

cd ../front
npm install
npm run dev
```

Abre tu navegador en: http://localhost:5173
